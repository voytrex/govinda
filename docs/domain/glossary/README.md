## How to Use

### For End Users

1. Choose your preferred language
2. Terms are organized alphabetically
3. Each term includes:
   - Definition
   - Translations in other languages
   - Related terms

### For Developers

1. German glossary is the primary reference (Swiss business standard)
2. Terms include code references where applicable
3. Use for consistent terminology in UI and documentation
